Kyomu

A lightweight, modern, self-hosted comic reader. Built for speed, designed for immersion.

⭐ GitHub Get Started

Everything you need

📚

Smart Library

Auto-scan CBZ, CBR, PDF and image folders. Supports Kapowarr & Mylar3 directory structures.

📖

Immersive Reader

Page-by-page, vertical scroll, double-page. Night mode, bookmarks, brightness control.

🔍

Discover & Download

Browse ComicVine catalog. One-click download via Kapowarr or Mylar3. Like Seerr, but for comics.

👤

Multi-Profile

Netflix-style profiles. Each user has their own reading progress and preferences.

🎨

Beautiful Design

Dark & light themes. Customizable accent colors. Responsive, touch-first interface.

⚡

Lightweight

Built with Next.js and SQLite. Minimal resource usage compared to Java-based alternatives.

Get started in seconds

Just Docker and your comics folder.

docker compose up -d
