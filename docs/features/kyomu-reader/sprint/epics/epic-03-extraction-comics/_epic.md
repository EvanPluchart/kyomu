# Epic 3 : Extraction de comics

## Objectif
Extraire les pages d'images depuis les differents formats de comics (CBZ, CBR, dossiers d'images) et les servir via API.

## Stories
- 3.1 — Implementer l'extraction des pages CBZ (M)
- 3.2 — Implementer l'extraction des pages CBR via binaire unrar (M)
- 3.3 — Implementer le support des dossiers d'images (S)
- 3.4 — Creer la factory d'extracteurs et les API Routes de pages (M)
