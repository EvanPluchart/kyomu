# Epic 6 : Lecteur de comics

## Objectif
Construire le lecteur de comics avec navigation page par page, gestes tactiles, progression, navigation inter-volumes et modes de lecture alternatifs.

## Stories
- 6.1 — Implementer le lecteur de base (affichage page par page) (L)
- 6.2 — Implementer les gestes tactiles (swipe, pinch-to-zoom) (L)
- 6.3 — Implementer le suivi de progression de lecture (M)
- 6.4 — Implementer la navigation inter-volumes (S)
- 6.5 — Implementer le mode de lecture vertical (webtoon) (M)
- 6.6 — Implementer le mode double page et le sens de lecture (M)

## Nombre de stories : 6
## Phase : 4
