# Epic 5 : Interface bibliotheque

## Objectif
Construire l'interface utilisateur de la bibliotheque : navigation, grille de series, detail, recherche et page d'accueil.

## Stories
- 5.1 — Creer le layout principal et la navigation (M)
- 5.2 — Implementer la grille de series (M)
- 5.3 — Implementer la page de detail d'une serie (M)
- 5.4 — Implementer la recherche et les filtres (M)
- 5.5 — Implementer la page d'accueil (M)
